Icey APT Repository

This path hosts the static APT repository for icey-server.

Canonical base URL: https://apt.0state.com/icey

Install the signing key:

curl -fsSL https://apt.0state.com/icey/icey-archive-keyring.gpg | sudo tee /usr/share/keyrings/icey-archive-keyring.gpg >/dev/null

Add the repository:

curl -fsSL https://apt.0state.com/icey/icey-server.list | sudo tee /etc/apt/sources.list.d/icey-server.list >/dev/null
sudo apt update
